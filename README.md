# Music Similar Finder

Aplicativo web para buscar músicas parecidas com base em artista, gênero, BPM, tom e ano.

## Como usar

Abra o arquivo `index.html` em um navegador ou sirva a pasta com qualquer servidor estático.

Exemplo com Python:

```bash
python -m http.server 8000
```

Depois acesse: http://localhost:8000

## Funcionalidades

- Busca por músicas
- Filtros por estilo, artista, BPM, tom e ano
- Sugestão de músicas semelhantes
- Funciona como PWA
